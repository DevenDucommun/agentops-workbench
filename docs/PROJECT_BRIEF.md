# Project Brief

## Name

AgentOps Workbench

## One-Liner

Local observability, risk analysis, and audit reporting for AI coding-agent sessions.

## Positioning

This is not another coding agent. It is the system of record for what an agent did.

## Differentiator

Most AI coding tools optimize generation. AgentOps Workbench optimizes trust, reviewability, and operational control.

## Ideal Demo

1. Run an AI agent on a coding task.
2. Ingest the session into AgentOps Workbench.
3. Produce a report showing:
   - files changed
   - commands run
   - tests executed
   - risky actions
   - missing evidence
   - final outcome
4. Use the report to decide whether the work is safe to merge.

## Good First Public Screenshot

A terminal command producing a clean Markdown audit report, plus a short report excerpt in the README.
